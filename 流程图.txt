
FlvLoader

构造函数初始化
  ├─ 保存 onData 回调
  ├─ 初始化 reader 为 null
  └─ destroyed 设置为 false

加载 FLV 文件（loadFlv）
  ├─ fetch 请求 URL
  ├─ 初始化 reader
  ├─ 如果已销毁，调用 abort 中止加载
  ├─ 检查 HTTP 状态码
      ├─ 状态有效
      │   ├─ 调用 onheader 处理响应头
      │   ├─ 如果重定向，调用 on302
      │   ├─ 调用 onopen 表示连接已打开
      │   └─ 调用 pump 开始读取流数据
      └─ 状态无效，调用 onerror 处理错误

读取流数据（pump）
  ├─ 调用 reader.read() 读取数据
  ├─ 检查是否读取结束
  │   ├─ 已结束：记录日志
  │   └─ 未结束：
  │       ├─ 将数据传递给 onData 回调
  │       └─ 递归调用 pump 继续读取数据
  └─ 捕获读取过程中的错误并调用 onerror

销毁加载器（destroy）
  ├─ 设置 destroyed 标志为 true
  ├─ 调用 abort 中止流读取
  └─ 调用 reset 重置加载器

重置加载器（reset）
  ├─ 将 reader 设置为 null
  └─ 将 destroyed 标志复位为 false

中止加载（abort）
  ├─ 检查 reader 是否存在
  ├─ 调用 reader.cancel() 取消流读取
  └─ 设置 reader 为 null
